# RPG Live Editor (RLE)

A modern runtime editor and debugger for RPG Maker games.

[ Screenshot ]

## Features

- Runtime Editing
- Live Translation
- Dialog Tools
- Variable Monitor
- Switch Editor
- Battle Tools
- Inventory Editor
- Lock State
- Runtime Event Stream
